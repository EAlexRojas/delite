Getting started with delite

Delite is providing you with a core UI infrastructure for building multi channel, enterprise class Web Components to be used in Web & Mobile Hybrid applications

xxx

Install Delite Generator

Start with installing Node.js, then you are ready to install Yeoman scaffolder and our delite generator. 
			{% highlight bash %}
$ npm install -g yo
$ npm install -g delite-element-generator
			{% highlight %}

Scaffold a New Project

Just ask Yeoman to scaffold a new element in your package directory by answering a simple set of questions. 
		
			$ mkdir -p custom && cd $_
			$ yo delite-element

Run it

Once your component has been generated you can run a generated sample using it. 
	
		$ bower install deliteful

Hack it

The default scaffolded element is just a skeleton you are willing to hack on. 

	$ vi CustomElement.js

Alternatively

Alternatively you could have directly installed delite and build your element from scratch instead of using Yeoman. 

	$ npm install -g bower
	$ bower install delite
	$ mkdir -p custom && cd $_
	$ vim CustomElement.js

Go further

Once you have created your own element you might want to mix it with other existing elements from the project and build your application from there. 
		
			$ npm install generator-deliteful-app
			$ yo deliteful-app
